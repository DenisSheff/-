# Home tasks

## Task 25
*Напишите цикл, который принимает на вход
два числа (A и B) и возводит число A в натуральную
степень B.*

### Example: 

3, 5 -> 243 (3⁵)

2, 4 -> 16

## Task 27
*Напишите программу, которая принимает на
вход число и выдаёт сумму цифр в числе.*

### Example: 

452 -> 11

82 -> 10

9012 -> 12
